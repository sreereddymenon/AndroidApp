# Android Open Weather App
This Open Weather App is  Built With
Jetpack Compose for UI
MVVM Architecture
Retrofit (for API Calling).
Hilt Dependency Injection.
Kotlin Coroutines.
StateFlow
Sealed Class


It uses https://home.openweathermap.org/
